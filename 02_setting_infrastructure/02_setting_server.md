# 「シールマニア」サイト構築手順書 (ConoHa WING編)
ConoHa WINGサーバーにて2つ目のWebサイト（シールマニア）を立ち上げるための操作手順
→ after-naviと同じ設定で作成する。

---

## 事前準備・前提
* ConoHa WINGのコントロールパネルにログインする
* ドメイン取得費用（数百円〜数千円程度）は必要になる。

---

## 手順1：新しいドメイン追加
サイトのURLとなるドメインを取得・設定する

1. 管理画面左側の青いメニューから **「サーバー管理」** をクリックします。
2. その右に出るサブメニューから **「ドメイン」** をクリックします。
3. 画面右上の **「＋ドメイン」** ボタン（青色）をクリックします。
4. **「新規ドメインを追加」** を選択し、**「ドメイン取得」** を選びます。
5. 検索バーに希望の文字列（下記推奨案を参照）を入力し、検索します。
6. 「カートに追加」→「次へ」と進み、決済を完了させます。

### 💡 【シールマニア】ドメイン名の推奨案
サイトの機能（検索・場所探し）や信頼性を重視した以下のドメインがおすすめです。

* **機能・検索重視（★推奨）:**
    * `seal-navi.com`
    * `seal-map.com`
    * `seal-search.com`
* **ブランド名重視:**
    * `seal-mania.com`

→SEOの基準を重視する。ドメインの空き状況も鑑みて、`seal-search.com`にする。

---

## 手順2：WordPressをインストールする

取得したドメイン領域に、WordPress本体をインストールします。

1. 管理画面左側の青いメニューから **「サイト管理」** をクリックします。
2. 画面左上の **「切り替え」** ボタン（現在のドメイン名が表示されている箇所）をクリックします。
3. **手順1で追加した新しいドメイン** を選択して切り替えます。
   * ※リストに表示されない場合は、数分待ってからページを再読み込みしてください。
4. 画面右上の **「＋WordPress」**（または＋サイト）をクリックします。
5. **「WordPress」** タブを選択し、以下の情報を入力して **「保存」** をクリックします。

### 📝 設定項目の入力指針

| 項目 | 入力内容・推奨値 |
| :--- | :--- |
| **バージョン** | 最新バージョン（デフォルトのまま） |
| **URL** | `www` なし（推奨） |
| **サイト名** | **`シールマニア｜ボンボンドロップ・シールの販売場所・入荷情報まとめ`**<br>※SEO対策のため、主要キーワード（商品名・場所・入荷）を含めます。 |
| **メールアドレス** | 管理用メールアドレス（受信可能なもの） |
| **ユーザー名** | 半角英数字（推測されにくい文字列を推奨） |
| **パスワード** | 半角英数字記号（強固なものを設定） |
| **データベース** | データベース名・ユーザー名は自動入力のままでOK。<br>パスワードは管理用に控えておく。 |

## 手順3：動作確認と初期設定

インストール完了後、以下の点を確認する」

### 1. SSL（https）化の待機
サイト作成直後、アクセスすると「保護されていない通信」等の警告が出ることがあります。
* **対応:** ConoHa側で「無料独自SSL」が自動設定されるまで **30分〜1時間ほど待ちます**。
* **確認:** 「サイト管理」>「サイトセキュリティ」>「無料独自SSL」が「利用中」になれば完了です。
