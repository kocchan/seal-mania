# 「シールマニア」事業構想・要件定義書

## 1. 事業概要
**シール交換ブーム（特に「ボンボンドロップ」等）における、販売・入荷情報の集約プラットフォーム。**

実店舗・通販の公式情報に加え、SNS上の目撃情報を「都道府県別・商品別」に構造化して提供する。情報の「速報性（フロー）」だけでなく、蓄積データに基づく「傾向分析（ストック）」を強みとし、ユーザーの購買行動を支援するWebメディアである。

## 2. 事業目的（Mission）

### 外向き（ユーザー価値）
* **「今どこで買えるのか」の可視化**
    * 分散する情報を集約し、最も分かりやすい情報ハブとなる。
* **保護者・ライト層の負担軽減**
    * SNSを常時監視しなくても、安心して購入判断ができる環境を提供する。

### 内向き（運営方針）
* **再現性のある「副業Webメディアモデル」の構築**
    * 低コスト、1人運営、半自動化で回る仕組みを作る。
* **将来的な横展開**
    * 他シール、文房具、コレクション商材へ展開可能な基盤（システム・ノウハウ）を確立する。

---

## 3. 提供価値（Value Proposition）

### 3.1 全体定義
単なる「情報の羅列」ではなく、**ユーザーの「意思決定（買いに行く／待つ／諦める）」を支援する判断材料**を提供する。

### 3.2 ユーザー状態別の解決策
| ユーザーの状態 | Before（課題） | After（提供価値） |
| :--- | :--- | :--- |
| **探している** | SNSを遡るのが手間、情報の新旧が不明 | 最新情報が1ヶ所に集約。発生日時が明確で**「探す時間」そのものを削減**。 |
| **迷っている** | 行って売っていないリスク、どこが狙い目か不明 | 地域・店舗の入荷傾向（ストック情報）から、**「不確実性」を下げて行動できる**。 |
| **初めて触れる** | 何が流行りか、どこで買うのが普通か不明 | シリーズ一覧・定価・販売場所の基本情報があり、**参入ハードルが下がる**。 |

### 3.3 情報整理の価値
* **時系列の整理:** いつ入荷したか明確で、古い情報に惑わされない。
* **地域の整理:** 都道府県単位で絞り込め、自分に関係ある情報へ即座にアクセスできる。
* **商品の整理:** シリーズ単位で俯瞰でき、再入荷の傾向が掴める。

### 3.4 信頼性と安心感
* **情報の可視化:** 「公式発表」「目撃情報」「未確認」を明確に区別する。
* **断定しない誠実さ:** 「売っている」ではなく**「◯月◯日時点で目撃情報あり」**というスタンスを貫き、誤情報による失敗体験
